# 粉丝列表

列出特定用户的关注者:

GET api/v2/users/:user/followers
1

列出授权用户的关注者:

GET api/v2/user/followers
1

参数

名称 类型 描述
limit Integer 获取条数，默认 20
offset Integer 翻页偏移量，默认 0

# 列出用户正在关注的人

列出一个用户正在关注的人：

GET /users/:user/followings
1

列出用户授权用户正在关注的人

GET /user/followings
1

# 参数

名称 类型 描述
limit Integer 获取条数，默认 20
