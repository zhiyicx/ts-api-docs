# 赞的人列表

1

# Parameters

名称 类型 描述
limit Integer 获取条数，默认 20
after Integer id 获取之后数据，默认 0

字段 描述
id 赞 ID
user_id 点赞用户 ID
target_user 接收赞用户 ID
likeable_id 赞的资源 ID，配置 likeable_type 表示不同资源
likeable_type 赞的资源类型。
created_at 点赞时间
updated_at 更新时间
user 点赞的用户资料，结构参考 「用户信息」接口说明。
