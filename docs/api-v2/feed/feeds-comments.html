# 发布评论

POST /feeds/:feed/comments
1

参数

Name Type Description
body String 必填，评论内容.
reply_user Integer 可选， 回复某个用户.

响应

Status: 201 Created
1

响应体

{
    "message": [
        "操作成功"
    ],
    "comment": {
        "user_id": 1,
        "reply_user": 0,
        "target_user": 1,
        "body": "我是第三条评论",
        "commentable_type": "feeds",
        "commentable_id": 1,
        "updated_at": "2017-07-20 08:53:24",
        "created_at": "2017-07-20 08:53:24",

}
