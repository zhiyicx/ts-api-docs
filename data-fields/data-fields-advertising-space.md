广告位列表

| 字段 | 类型 | 说明 |
| :- | :- | :- |
| id | int | 自增id |
| channel | varchar | 广告位所在频道 |
| space | varchar | 广告位位置标示 |
| alias | varchar | 广告位别名 |
| allow_type | varchar | 允许的广告类型 |
| format | text | 广告数据格式 |
| rule | text | 广告位表单验证规则 |
| message | text | 广告位表单验证提示  |
| created_at | timestamp | 创建时间 |
| updated_at | timestamp | 更新时间 |
